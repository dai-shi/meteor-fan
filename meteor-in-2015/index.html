2015年のMeteor総まとめ

01 January 2016 | ニュース

Meteorの公式ブログに2015年の活動のまとめが載っていますので紹介します。

Meteorプラットフォーム

  • Meteor 1.0.3.1リリース。パッケージまわりのコマンド強化。
  • Meteor 1.0.4リリース。MongoDB 2.6と3.0のサポート。テンプレートレベルのSubscribeの追加。
  • Meteor 1.1リリース。Windowsのサポートの追加。

  • Meteor 1.2リリース。React/Angularサポートの追加。ES2015の導入。

  • Meteor Galaxyのリリース。開発者向けエディションも。

  • GitHubで3万スター獲得し、Railsを抜いてトップに。
  • Angular Meteor 1.2.0リリース。Angular-Meteor 1.3リリース。Blaze互換性アップ。
  • Meteor 1.3ベータアナウンス。

コミュニティ・イベント

  • 21回のDevShopがSF/London/New Yorkで開催。YouTubeで視聴可能。
  • Meetupに60都市と約4万メンバーの登録。
  • Student Hackathonに15以上の大学から参加。
  • ニュースサイトMeteorWeeklyの開始。
  • Global Distributed Hackathonに90都市より1000以上の開発者の参加。
  • Meteor Space Campの開催。

学習リソース・オープンソースプロジェクト

  • 公式チュートリアル Blaze / Angular / React
  • WhatsAppクローン Ionicチュートリアル
  • Courseraコース
  • 書籍Meteor in Action
  • Meteor Guide作成中
  • React in Meteor
  • Gravity SNSクローン
  • LevelupTutsの動画チュートリアル Meteor for Everyone / Intermediate Meteor
  • Rocket.Chat Slackクロ一ン
  • Wekan Trelloクローン
  • GraphQLパッケージ
  • The Meteor Chefの17レシピと43スニペット
  • From Blaze to Reactのブログ記事

会社(MDG)

  • $20Mの追加調達 日本語記事
  • 有償開発者サポート 日本語記事
  • ガートナーのレポート
  • CEOのYouTubeトークが4万試聴数
  • 68のパートナー登録
  • Scholarship Program 日本語記事

感想

2015年は盛り沢山でした。だいぶ振り回された感もありますが。
2016年のさらなる発展に期待しましょう。個人的にはMeteor Guideに注目。
